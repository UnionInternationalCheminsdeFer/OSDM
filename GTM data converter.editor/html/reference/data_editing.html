Data editing

    Data are presented in a tree structure where the basic branches are created automatically. Tree branches can be edited by the functions of the context menue on a tree branch to:

    • add child data
    • add sibling data
    • delete data
    • copy/paste data
    • validate data
      • .

    The tree leaves can be edited in the properties window which is displayed when a tree branch is selected. Text and numeric values can be entered directly, whereas codes are added from listboxes. In case multiple entries are possible a multi-select diaolog will pop up.

    Note: Data objects contained in another object are edited via the context menue (e.g. multilanuage text data object within the list of texts), whereas references to such a text object are edited as a leave in the property view via a selection list. (e.g. referenzing a text data object within a fare).
