## Modified Objects

- `Fare`: required constraint removed for property `requiredCards`

- `ActualPassengerType`: changed `COMPANION_DOG` for `ACCOMP_DOG` (see code

- `TransportableType`: items consolidated to: `PET`, `DOG`, `ACCOMP_DOG`,
  `BICYCLE`, `LUGGAGE`, `PRAM`, `CAR`, `MOTOCYCLE`, `TRAILER` (see code lists)
